freee のアカウントを作る 🏦

⏱ 約10分 🤝 準備ステップ

まずは伝票が起票される「箱」をご用意ください。クラウド会計freeeの無料アカウントを作成します。すでにfreeeのアカウントをお持ちの方はこのステップは飛ばしてOKです。

💡 これで何がラクになる？
freeeは「取引」を入れると、複式簿記の仕訳を自動で組み立ててくれるクラウド会計です。あとでAIがここに直接書き込めるようになります。
📋 手順
  1. freee公式サイト を開く
  2. 「無料で始める」からメールアドレスで登録
  3. 確認メールのリンクを踏んで本登録
  4. 事業所名（屋号や会社名／個人なら自分の名前でOK）を入力
  5. ログイン後、ダッシュボードが表示されれば完了です！
🔒 安全メモ
パスワードの使い回しはお控えください。ログイン情報やトークンが映る画面は📸スクリーンショット・画面共有をしないようご注意ください。
🛠 迷ったとき・トラブル
  • 確認メールが来ない → 迷惑メールフォルダをご確認ください。数分待っても来ない場合は再送してください。
  • 事業所はあとで変えられる？ → はい、設定からいつでも変更可能です。気軽に進めていただいて大丈夫です。
  • 有料プランに入る必要は？ → freeeは新規登録から30日間のお試し期間であれば、口座連携・freee MCP（API連携）を無料でご利用いただけます。ただしお試し期間が終了し有料プランをご契約いただかないと、口座連携・API連携は自動的に解除・停止されます。今日の体験のみであれば問題ありませんが、実務で継続利用される場合は有料プランのご契約が必須です。

銀行口座・クレジットカードを連携する 🏧

⏱ 約10分 🤝 準備ステップ

freeeに口座やカードをつなぐと、明細が自動で取り込まれます。これが「AIが自動で起票する」体験の元になります。すでに口座連携が済んでいる方はこのステップは飛ばしてOKです。

💡 これで何がラクになる？
口座・カードを連携すると、入出金明細が自動でfreeeに届くようになります。あとはAIがその明細を見て「これは消耗品費っぽいですね、登録しますか？」と提案し、起票するという流れが作れます。
📋 手順
  1. freeeにログインし、左メニューの「口座」（または「自動で経理」）を開く
  2. 「新しく口座を登録」→ 連携したい銀行・クレジットカード会社を検索
  3. 各金融機関のネットバンキングID・パスワードを入力してログイン許可
  4. 同期が始まったら完了です。初回は明細の取り込みに数分〜数十分かかることがあります
🔒 安全メモ
ネットバンキングのID・パスワードはfreee社の正規連携画面以外には絶対に入力しないでください。連携作業中の画面は📸スクリーンショット・画面共有をしないようご注意ください。
🛠 迷ったとき・トラブル
  • 連携したい銀行が見つからない → 銀行名・支店名で再検索してください。それでも出ない場合はCSVでの明細取込に切り替え可能です（当日サポートいたします）。
  • 明細がなかなか出てこない → 初回同期は時間がかかることがあります。10〜20分待っても出ない場合は「今すぐ同期」を再実行してください。
  • 連携自体に料金はかかる？ → 新規登録から30日間のお試し期間中は無料で口座連携をご利用いただけます。お試し期間後も連携を続けるには有料プランのご契約が必要です（未契約のままだと連携が解除されます）。
  • すでに連携済みでテスト用の明細がない → ステップ④に「テスト取引を手動で起票してもらう」コピペ文もご用意していますので、そちらをお使いください。

Claude Code をインストールする 💻

⏱ 約15分 ⚡ セットアップ

次は、AIに作業をお願いするための道具「Claude Code」を自分のPCに入れるよ。インストール手順は事前に共有しているUnikoukokunさんの課題ページにまとまっているので、そちらを見ながら進めてね。

💡 これで何がラクになる？
Claude Codeは、AIに自然な言葉で指示できるデスクトップアプリ。「この請求書を登録して」みたいな依頼を、そのまま実行に変えてくれる相棒だよ。ターミナル（黒い画面）は不要、デスクトップアプリだけで完結するよ。
📋 進め方
  1. こちらのページを開く → Unikoukokunさんの課題ページ（課題1-1）
  2. 「課題1-1」の手順に沿って、Mac / Windowsどちらか自分のPCに合う方でインストール
  3. アカウント作成は「Googleで続ける」から、ふだん使っているGoogleアカウントを選ぶだけでOK
  4. Claude を課金（Proプラン・月額制／約22ドル＝3,300円前後）する。課金しないとClaude Codeは動かないので、ここは必須
  5. （Gitが必要と案内された場合のみ）Mac は GitHub Desktop、Windows は git-scm.com からGUIインストール。インストール後はPCを再起動するかアプリを開き直す

Claude Codeアプリを開いて「こんにちは」と話しかけてみて、返事が来れば成功！

🔒 安全メモ
クレジットカード情報の入力画面・ログイン画面は📸スクショ・画面共有しないでね。
🛠 迷ったとき・トラブル
  • 課金しないと使える？ → 使えない。無料プランのままだとClaude Codeは動かないよ。月額のProプラン（年額ではなく月額を選ぶこと）に入ってね。
  • Googleアカウントがない → メールアドレスでの新規登録もできる。当日サポートするね。
  • Gitを案内されたけどよく分からない → アプリの案内に従ってGUIインストールするだけでOK。コマンド入力は不要。
  • インストールしたのに反応しない → アプリを完全に閉じて開き直す。それでもダメならPCを再起動。

freee MCPサーバーを接続して、自動起票を体験する 🔌

⏱ 約20分 🚀 本番・体験

いよいよ本番！ Claude Code と freee を「MCP」でつなぐと、AIがあなたの代わりにfreeeへ伝票を起票できるようになるよ。

💡 MCPってなに？
MCP（Model Context Protocol）は、AIと外部サービスをつなぐ“共通プラグ”。これを挿すと、Claude Codeが「freeeを操作する手」を手に入れるイメージだよ。
📋 手順①：freee MCPを追加する（プラグイン方式）
  1. Claude Codeアプリを開き、下の2つを会話画面（チャット欄）にそのままコピペして送信 ↓（ターミナルは不要。freee公式が配布している正規のプラグインを使う方法で、OAuthの個別設定もいらない）
📋 ① freee公式プラグインのマーケットプレイスを追加
/plugin marketplace add freee/freee-mcp
📋 ② freee MCPプラグインをインストール
/plugin install freee-mcp@freee-mcp-marketplace
📋 手順②：認証する
  1. インストール後、freeeのログイン画面が自動的にブラウザで開く
  2. freeeにログイン →「許可する」をクリック
  3. 連携したい事業所を選択（ステップ①で作った事業所を選ぶ）
  4. ターミナルに戻り 認証OK の表示が出れば接続成功！🎉

※ 権限はログイン中のfreeeユーザーと同じ範囲に限定されるよ。

🏃 メイン体験：口座連携した明細から、AIに自動で起票してもらう
freeeに同期されている未処理の取引明細（自動で経理）を1件見せて。
内容を踏まえて、勘定科目の候補を提案してね。
私がOKと言ったら、その内容でfreeeに登録して。

Claudeが明細を確認 → 勘定科目を提案 → あなたが「OK」と返す → freee側に取引が自動で起票される。これが今日のメインゴールだよ✨ freeeの「取引」一覧を開いて、登録されているか見てみよう！

🙋 ステップ②を飛ばした／同期明細がまだない人はこちら

口座連携がまだ、または明細の取り込みが間に合っていない場合は、代わりにこちらをコピペしてね。テスト取引で同じ流れを体験できるよ。

📋 テスト用：AIに伝票を起票してもらう
freeeに、本日付で「消耗品費 1,100円（税込）／現金」の取引を1件、テストで起票して。
登録前に内容を見せて、OKしたら登録してね。
🔒 安全メモ
  • 必ず「登録前に内容を見せて」と添えて、確認してから登録させる癖をつける。
  • 最初は少額・少件数で試すのが安心。
  • 認証画面・トークンが映る画面は📸撮らない・共有しない。
🛠 迷ったとき・トラブル
  • 🟥 認証はOKなのに「事業所が見つからない」 → freeeにログインしている事業所と、認証時に選んだ事業所が同じか確認。
  • 🟥 起票したのにfreeeに出ない → 取引一覧の日付フィルタを確認。今日の日付に絞って再表示。
  • プラグインが見つからない／インストールに失敗する → 会話画面で /plugin marketplace list と送って追加されているか確認。再度①からやり直す。
  • 権限エラー → freee側でアプリ連携を「許可」したか再確認。一度 /mcp と送ってfreeeの接続状態を確認するのもアリ。
  • 同期明細が1件も出てこない → ステップ②の口座連携がまだの可能性。上の「テスト用」コピペで体験を進めてOK。

時間がある人向け：AI事務所テンプレート一式を入れる

⏱ 約1〜1.5時間 🌟 任意（チェック対象外）

ここまでで「自動起票デビュー」は完了。時間に余裕がある人だけ、ここから先に進んでみよう。畠山から配布している「AI事務所テンプレート一式」を使うと、Claude Codeにあなた専用のキャラ・事務所ルール・安全ロック・記憶機能を一気に持たせられるよ。

💡 何が手に入る？
  • ① md設定：AIの名前・性格、事務所のルール（CLAUDE.md）
  • ② セキュリティ：.env誤読み取り・外部送信・rm -rf事故を防ぐロック
  • ③ メモリ：「前にも言いましたよね」を実現する記憶の仕組み
  • ④ Hooks：長い会話・要約後も前提を忘れない仕掛け
📖 詳細解説はこちら（任意）
📋 手順
  1. 下のリンクからテンプレート一式（フォルダ）をダウンロードし、デスクトップなど分かりやすい場所に保存 → AI事務所テンプレート一式（Googleドライブ）
  2. Claude Codeで、保存したフォルダを開く
  3. 下のコピペ文を会話画面に貼って送信（{{ }}の部分は自分の情報に書き換えてね）↓
📋 Claude Codeに丸ごと渡すプロンプト
このフォルダには、Claude Codeのセットアップ一式（4点セット）が入っています。
セットアップ/README.md を読んで、書かれている順番（①md→②セキュリティ→③メモリ→④Hooks）の通りに、
私のPCへの設置を進めてください。

私は{{税理士／会計士／社労士／行政書士／弁護士／その他}}で、
事務所名は「{{あなたの事務所名}}」です。

進め方のルール：
- 各フォルダのREADMEに書いてある手順に沿って進めてください
- カスタマイズに必要な情報（名前・呼び方・キャラの性格など）は、その都度1問ずつ私に聞いてください
- 既存ファイルの上書き・バックアップなど、確認が必要な場面では必ず先に私に確認してください
- 1つのフォルダの設置が終わったら、「①が終わりました。②に進みますか？」のように区切って報告してください
- 全部終わったら、最後に「動作確認の手順」を案内してください

あとはClaude Codeが質問してくるので、答えながら進めるだけ。「コードが分からないから無理」と思う必要はなし。一気に4つ不安なら「まずは①だけ進めてください」と区切ってお願いしてもOK。

🔒 安全メモ
既存ファイルの上書きが発生する場面ではClaude Codeが確認を挟む設計になってる。よく分からないまま「はい」を連打せず、聞かれたら内容を読んでから答えてね。
🛠 迷ったとき・トラブル
  • 設定したのに反映されない → Claude Codeを完全に再起動（タスクトレイに残っていないかも確認）。
  • .claude フォルダが見つからない → 隠しフォルダなので「隠しファイルを表示」をON。一度Claude Codeを起動すると自動で作られる。
  • このステップは今日できなかった → 問題なし。次回以降のブートキャンプで進めればOK。
