🔤 基本のことば

まずはここから。会話の土台になる言葉です。

生成AIせいせいエーアイ

文章・画像・プログラムなどを「作り出せる」人工知能のこと。ChatGPTやClaudeが代表例です。決まった答えを返す従来のソフトと違い、指示に応じて中身を毎回組み立てます。

Claudeクロード

Anthropic（アンソロピック）社が作っている生成AIの名前。ChatGPTの同業他社にあたります。この講座で使うのはこのClaudeです。

Claude Codeクロードコード

Claudeに「会話」だけでなく「PC上の作業」まで任せられるデスクトップアプリ。ファイルを作る・freeeに起票する・資料を読むといった実作業を、チャットで頼むだけで実行してくれます。

たとえるなら：ChatGPTが「電話相談できる相手」なら、Claude Codeは「事務所に来て手を動かしてくれるスタッフ」です。
プロンプトprompt

AIへの指示文のこと。「この請求書をfreeeに登録して」もプロンプトです。特別な書き方は不要で、スタッフへの依頼メモと同じ感覚で書けば伝わります。

チャットchat

AIとやり取りする会話画面のこと。LINEのトーク画面のイメージです。課題ページの「チャットに貼り付けて送信」は、この画面の入力欄に文章を貼ることを指します。

コピペコピー＆ペースト

文章をコピーして貼り付けること。Windowsは Ctrl+C（コピー）→ Ctrl+V（貼り付け）、Macは Command+C → Command+V。課題ページの「コピー」ボタンを押せば自動でコピーされます。

📄 ファイルまわり

「作ってもらった資料がどこにあるか分からない」を防ぐ言葉たち。

マークダウン（.md）Markdown・エムディーファイル

見出しや箇条書きを簡単な記号で表す「メモの書き方のルール」、およびその形式で書かれたファイル（拡張子 .md）のこと。# が見出し、- が箇条書きを意味します。Claude Codeが議事録やメモを作るときの標準形式で、Wordは不要。Claude Codeアプリの中でそのまま読めます。

たとえるなら：Wordのような「アプリ」ではなく、「速記のルール」に近いもの。ルールさえ知っていればどのPCでも読み書きできます。
背景：2004年生まれの書き方で、エンジニアの世界では20年来の共通語。AIが読み書きしやすい形式のため、生成AIの普及とともに一般にも広がりました。
拡張子かくちょうし

ファイル名の末尾につく「.pdf」「.xlsx」「.md」などのこと。ファイルの種類（何のアプリで開くものか）を表します。

フォルダ／ディレクトリfolder / directory

ファイルの入れ物のこと。「ディレクトリ」は同じ意味のIT用語です。Claude Codeでは「どのフォルダを開いて作業するか」が大事で、メモリなどの設定はフォルダごとに保存されます。

ZIP／解凍ジップ／かいとう

複数のファイルを1つにまとめて圧縮したファイルがZIP。中身を取り出すことを「解凍（展開）」と言います。配布テンプレートをダウンロードすると、この形式のことがあります。

CSVシーエスブイ

データをカンマ（,）で区切っただけのシンプルな表形式ファイル。Excelで開けます。freeeやMF会計への明細取込・一括登録でよく使われます。

💻 Claude Codeのことば

アプリの設定・カスタマイズで出てくる言葉たち。テンプレート導入時にも登場します。

インストールinstall

アプリやソフトを自分のPCに入れて使えるようにすること。逆に消すことは「アンインストール」と言います。

ターミナル（黒い画面）terminal

文字だけでPCを操作する画面。エンジニアがよく使うものですが、この講座ではターミナルは使いません。Claude Codeデスクトップアプリのチャットだけで完結します。

コマンドcommand

PCへの命令文のこと。「このコマンドを実行してもよいですか？」と聞かれたら、Claudeが命令文を代わりに実行しようとしている場面です。内容を読んで許可すればOK。自分で打ち込む必要はありません。

プラグインplugin

アプリに後づけできる拡張機能のこと。第1回で入れる「freee MCPプラグイン」は、Claude Codeに「freeeを操作する機能」を後づけするものです。

たとえるなら：スマホに入れる「追加アプリ」のようなもの。本体はそのままに、できることが増えます。
CLAUDE.mdクロード・エムディー

Claude Codeが会話の最初に毎回読み込む「事務所ルールブック」のファイル。AIの名前・性格、やってよいこと・ダメなことをここに書いておくと、毎回言わなくても守ってくれます。テンプレート①で設定するものです。

背景：ファイル名はClaude専用ですが、「AIへの指示書をフォルダに置いておく」というやり方自体は、他のAIツールにも広がりつつある文化です。
メモリmemory

Claude Codeの「記憶帳」。会話をまたいで覚えておいてほしいこと（好み・ルール・顧問先情報など）をファイルとして保存する仕組みです。「前にも言いましたよね」が通じるようになります。テンプレート③で設定します。

背景：ChatGPTにも「メモリ」という同名の機能があります。「AIに長期記憶を持たせる」考え方は生成AI共通で、実現のしかたがツールごとに違うだけです。
hooks（フック）フックス

「◯◯のタイミングで、自動で△△する」という仕掛けのこと。たとえば「会話が長くなって要約された後も、前提ルールを読み直させる」といった自動化に使います。テンプレート④で設定します。

たとえるなら：「来客があったらお茶を出す」のような、条件つきの自動ルールです。
背景：「フック（引っかけて割り込む仕組み）」はプログラミングの世界の古くからの一般用語。Claude専用語ではなく、他のソフトの説明にも出てきます。
settings.jsonセッティングス・ジェイソン

Claude Codeの動作設定が書かれたファイル。テンプレートの「セキュリティ」「Hooks」で編集します。壊れると動作に影響するので、変更前に必ずバックアップを作るのがお約束です。

背景：「設定をJSONファイルに書く」やり方自体はIT一般の文化。settings.jsonというファイル名と、中に書ける項目のルールがClaude Code固有です。
スラッシュコマンド（/◯◯）slash command

「/」で始まる、Claude Code専用の短縮命令。ただし「/plugin ...」のような断片だけをチャットに貼っても動きません。課題ページのコピペ文をまるごと貼って、Claudeにお願いする形が確実です。

背景：「/」で命令する文化はSlackやDiscordなどのチャットツール共通。ただし使えるコマンドの中身はツールごとに違います。
再起動さいきどう

アプリやPCを一度完全に終了して、開き直すこと。設定やプラグインは開き直したときに読み込まれるため、「入れたのに反応しない」の大半は再起動で直ります。タスクトレイ（画面右下）やDockに残っていないかも確認を。

🧰 テンプレート導入のことば

配布している「AI事務所テンプレート一式」のセットアップで出てくる言葉たち。READMEやClaudeの報告に出てきたら、ここで確認してください。

READMEリードミー

「最初に読んでください（Read me）」という名前の説明書ファイル。フォルダを受け取ったら、まずREADMEを探して読む（またはClaudeに「READMEを読んで進めて」と頼む）のがITの世界のお約束です。

テキストファイル（.txt）テキスト

飾りのない「文字だけ」のファイル。メモ帳で開ける、いちばん基本の形式です。注意点がひとつ：拡張子が非表示の設定のままファイル名を変えると、知らないうちに「◯◯.md.txt」という二重拡張子になって読み込まれない事故が起きがちです。エクスプローラー／Finderで拡張子表示をONにしておくと安心です。

シェルスクリプト（.sh）エスエイチファイル

PCへの命令をまとめて書いた「台本」ファイル。テンプレートのセキュリティガードやhooksの実体（credential-guard.sh、session-start.sh など）がこれです。中身を読めるようになる必要はありません。「このスクリプトは何をするの？」とClaudeに聞けば日本語で説明してくれます。

たとえるなら：新人さんに渡す「この順番でやってね」という作業手順書。実行するのはPCです。
JSONジェイソン（.json）

プログラム用のデータの書き方のルール。settings.json の中身がこの形式です。カンマやカッコが1つ欠けるだけでエラーになる繊細な形式ですが、手で直す必要はなく、エラーメッセージごとClaudeに貼れば直してくれます。

背景：名前はJavaScriptというプログラミング言語に由来。いまでは言語を問わず、IT業界全体の共通データ形式になっています。
.claude フォルダドットクロード

Claude Codeの設定一式（settings.json・hooks・メモリなど）が入る専用フォルダ。テンプレート①〜④の設置先はここです。名前が「.」で始まるため、ふだんは隠れています（次の「隠しファイル」参照）。

隠しファイル／隠しフォルダかくしファイル

名前が「.」で始まるファイルやフォルダ（.claude、.env など）は、初期設定では画面に表示されません。「見つからない！」となったら、エクスプローラー／Finderの「隠しファイルを表示」をONにしてください。

.envドットエンブ

パスワードやAPIキーなどの秘密情報をまとめて書いておくファイル。絶対に人に見せない・AIにも不用意に読ませないのが鉄則で、テンプレート②のセキュリティロックは「.envの誤読み取り・外部送信」を防ぐためのものです。

たとえるなら：事務所の金庫の鍵束。だからこそ、うっかり触らせないロックを最初にかけます。
背景：Web開発の世界で広く使われてきた慣習的なファイル名。「秘密情報は.envに隔離して、共有物に含めない」が業界の常識になっています。
パス（フルパス）path

ファイルやフォルダの「住所」。C:\Users\hatakeyama\Documents\◯◯.md のような文字列です。「フルパスを教えて」と言われたら、この住所全体のこと。ClaudeにファイルのありかをQ&Aするときによく登場します。

Git Bash／bashギットバッシュ／バッシュ

bash（バッシュ）は .sh ファイルを動かす「実行係」。Macには標準で入っていますが、Windowsには無いため、Gitをインストールすると付いてくる「Git Bash」で代用します。テンプレート②と④が裏側で使うので、Windowsの方は先にGitを入れておきます。

permissions（allow / ask / deny）パーミッションズ

Claude Codeに対する「やってよい（allow）・毎回確認（ask）・禁止（deny）」の権限リスト。settings.json の中に書きます。テンプレート②は、危険な操作をこのdenyとaskに登録することで事故を防いでいます。

たとえるなら：新人スタッフへの職務権限規程。「押印は所長決裁」「シュレッダーは確認してから」を明文化するイメージです。
背景：「権限（パーミッション）」という考え方自体はセキュリティ業界の一般用語。allow / ask / deny の3段階で書く方式がClaude Code固有です。
rm -rfアールエム・マイナスアールエフ

フォルダを中身ごと、確認なしで完全削除する命令。ゴミ箱にも残りません。テンプレート②のロックが真っ先に防ぐ対象で、「rm -rf事故」と言ったらAIやスクリプトの誤操作による大量削除のことです。

背景：Macの土台にもなっているUnixという歴史あるOSから続く伝統的なコマンド。IT業界では「危険な命令」の代名詞として有名です。
実行権限（chmod +x）じっこうけんげん

「このファイルは実行してOK」という印。.sh ファイルはこの印が無いと動かず、「permission denied」というエラーになります。hooksが反応しないときの定番チェックポイント。Claudeに「実行権限を確認して直して」と頼めばOKです。

改行コード（CRLF / LF）かいぎょうコード

「改行」の内部的な表し方が、Windows（CRLF）とMac（LF）で違います。ZIPやドライブ経由で受け取った .sh ファイルが動かないときの定番原因。見た目では分からないので、Claudeに「改行コードを確認して、CRLFならLFに直して」と頼むのが早道です。

バックアップ（.bak）backup／ドットバック

変更前のファイルのコピー。ファイル名の末尾に「.bak」や日付（settings.json.bak_20260709 など）を付けて残します。設定をいじって壊れても、バックアップを元のファイル名に戻せば復旧できます。settings.json を変更する前は必ず作るのがテンプレートのルールです。

.gitignoreギットイグノア

「Gitの管理（履歴・共有）に入れないファイル」を並べたリスト。.env などの秘密情報を、うっかり履歴や共有に含めないための安全装置です。テンプレート②にひな形が入っています。

セッションsession

Claude Codeとの「1回の会話のまとまり」。新しいチャットを開始してから閉じるまでが1セッションです。「セッションを跨いで覚える」＝会話を新しくしても覚えている、という意味で、それを実現するのがテンプレート③のメモリです。

コンテキストcontext

AIが「いま頭に入れている文脈情報」のこと。会話の履歴や読んだファイルの中身がこれにあたります。コンテキストには容量の上限があるため、長い会話では古い内容が押し出されて「前提を忘れる」現象が起きます。

たとえるなら：打ち合わせ中の「短期記憶」。メモ（メモリ）に書かない限り、会議が終われば忘れます。
圧縮（/compact・コンパクト）あっしゅく

会話が長くなったとき、履歴を要約してコンテキストの空きを作るClaude Codeの機能。便利な反面、要約の過程で大事なルールが抜け落ちることがあります。それを自動で読み直させるのがテンプレート④のhooks（post-compact）です。

🔌 つなぐ・連携のことば

freeeとClaude Codeをつなぐときに出てくる言葉たち。第1回のメインどころです。

MCPエムシーピー（Model Context Protocol）

AIと外部サービス（freeeなど）をつなぐための「共通プラグ」の規格。これを挿すと、Claude Codeが「freeeを操作する手」を手に入れます。第1回で入れるfreee MCPは、freee公式が無料公開している正規のものです。

たとえるなら：家電のコンセントの規格。プラグの形が共通だから、どのメーカーの家電（サービス）でも同じ差込口（AI）につながります。
背景：もともとはClaudeの開発元Anthropic社が2024年に公開した規格。その後ChatGPTなど他社のAIも採用し、いまではAI業界の共通規格になっています。
APIエーピーアイ

ソフト同士がやり取りするための「プログラム専用の窓口」。人間はfreeeの画面をマウスで操作しますが、プログラムはAPIという裏口から直接データを読み書きします。画面操作より速くて確実です。

たとえるなら：銀行の「窓口」が人間用、「全銀システム」がプログラム用。APIは後者にあたります。
APIキー／トークンエーピーアイキー／token

APIを使うための「合鍵」にあたる長い文字列。これを知っている人は誰でもあなたの代わりに操作できてしまうので、スクリーンショットに写さない・チャットや掲示板に貼らない・人に教えない、が鉄則です。

認証／認可（許可する）にんしょう／にんか

「このアプリに、私の代わりにfreeeを操作することを許可します」という手続き。第1回でブラウザが開いて「許可する」を押すのがこれです。許可の範囲は、ログインしているfreeeユーザーの権限と同じ範囲に限定されます。

連携／同期れんけい／どうき

サービス同士をつないで、データが自動で流れる状態にすること。「口座連携」なら、銀行の入出金明細が自動でfreeeに届くようになります。「同期」は最新の状態に揃えることです。

クラウドcloud

データやソフトが自分のPCではなく、インターネット上のサーバーにある状態のこと。freeeが「クラウド会計」と呼ばれるのは、どのPCからログインしても同じデータが使えるからです。

ローカルlocal

クラウドの反対で、「自分のPCの中」のこと。「ローカルに保存」と言われたら、自分のPCの中に保存するという意味です。

ブラウザbrowser

インターネットのページを見るためのアプリ。Chrome、Edge、Safariなどのことです。freeeはブラウザで開くサービスです。

🛠 開発ツールのことば

画面の案内やエラーで出てきたときに「ああ、あれね」と思い出せればOKの言葉たち。使いこなす必要はありません。

Gitギット

ファイルの変更履歴を管理する道具。エンジニアの必需品ですが、この講座では「プラグインの取得などの裏方として必要になることがある」程度の関わりです。「gitが見つかりません」と言われたら、案内に沿って入れるだけでOK。

GitHubギットハブ

Gitで管理しているファイルを置いておける、インターネット上の倉庫サービス。freee MCPプラグインなどの配布元としても使われています。

Pythonパイソン

世界で最もよく使われているプログラミング言語のひとつ。第3回の「freeeオート経理（体験版）」はPythonで書かれています。自分で書ける必要はありません。動かすためにPCに入れておく、という位置づけです。

GUIジーユーアイ

マウスでクリックして操作できる、ふつうの画面のこと。「GUIインストール」は「黒い画面に文字を打つのではなく、画面の案内に沿ってクリックで入れられますよ」という意味です。

エラーerror

うまく動かなかったときにPCが出すお知らせ。赤い文字や英語で出ると身構えますが、エラー文をまるごとコピーしてClaudeに貼れば、原因と対処を日本語で説明してくれます。怖がらず素材として扱うのがコツです。

OCRオーシーアール

画像やPDFに写っている文字を読み取ってデータにする技術。第3回の「請求書PDFを読み取ってfreeeに起票」で活躍します。

💳 料金・プランのことば

お金まわりで迷わないための言葉たち。

Proプラン（課金）プロプラン

Claudeの月額有料プラン（月額制・約22ドル＝3,300円前後）。Claude Codeは無料プランでは動かないため、講座ではProプランへの加入が必須です。年額ではなく月額を選べばいつでもやめられます。

使用上限（レート制限）しようじょうげん

有料プランでも、一定時間にたくさん使うと「使用上限に達しました」と一時停止になる仕組み。故障ではありません。時間を置くと自動で復活します。休憩を挟むか、続きは後日でOKです。

お試し期間（トライアル）trial

freeeに新規登録すると使える30日間の無料期間。この間は口座連携もAPI連携も無料で使えます。期間終了後も続ける場合は有料プランの契約が必要で、未契約のままだと連携は自動的に解除されます。

サブスクリプション（サブスク）subscription
